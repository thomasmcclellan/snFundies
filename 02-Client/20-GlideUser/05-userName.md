# `userName`
## Return value:
| Type | Description |
|---|---|
| `string` | User name of the current user. |

---

## Description:
Returns the user name of the current user (i.e. gsmith02, not George Smith).

---

```js
var userName = g_user.userName;

alert('Current user = ' + userName);
```
