# `userID`
## Return value:
| Type | Description |
|---|---|
| `string` | `sys_id` of the current user. |

---

## Description:
Returns the `sys_id` of the current user.

---

```js
var userID = g_user.userID;

alert('Current user ID = ' + userID);
```
