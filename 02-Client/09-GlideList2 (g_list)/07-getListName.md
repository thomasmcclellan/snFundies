# `getListName()`
## Return value:
| Type | Description |
|---|---|
| `string` | The list name (usually the table name). |

---

## Description:
Returns the name of the list, which is usually the table name.

---

```js
var list = GlideList2.get(name),
    listName = list.getListName();
```
