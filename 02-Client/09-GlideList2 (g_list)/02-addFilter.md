# `addFilter(filter)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| filter | `string` | Query `string` condition to add. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Add a single term to the list query filter.

---

```js
g_list.addFilter('active=true');
```
