# `disableAttachments()`

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Prevents file attachments from being added.

  > This method is not available on the mobile platform.  If this method is run on a mobile platform, no action occurs.
