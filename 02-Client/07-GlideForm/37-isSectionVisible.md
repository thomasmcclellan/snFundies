# `isSectionVisible(sectionName)`

## Parameter(s):
| Name | Type | Description |
|---|---|---|
| sectionName | `string` | The section name. |

## Return value:
| Type | Description |
|---|---|
| `boolean` | Returns `true` when the section is visible; otherwise, `false`. |

---

## Description:
Returns `true` if the section is visible.
