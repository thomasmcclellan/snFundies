# `getFormElement()`

## Return value:
| Type | Description |
|---|---|
| `HTMLElement` | The `HTML` element for the form. |

---

## Description:
Returns the `HTML` element for the form.

  > This method is not available in mobile scripts or Service Portal scripts.
