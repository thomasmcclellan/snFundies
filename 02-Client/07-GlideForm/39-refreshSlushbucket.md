# `refreshSlushbucket(fieldName)`

## Parameter(s):
| Name | Type | Description |
|---|---|---|
| fieldName | `string` | Name of the slush bucket. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
You can update a list collector variable.

---

```js
g_form.refreshSlushbucket('bucket');
```
