# `isMandatory(fieldName)`

## Parameter(s):
| Name | Type | Description |
|---|---|---|
| fieldName | `string` | Name of the field. |

## Return value:
| Type | Description |
|---|---|
| `boolean` | Returns `true` if the field is required; otherwise, `false`. |

---

## Description:
Returns `true` if the field is mandatory.
