# `sortDescending(field)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| field | `string` | The field used to sort the list. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sorts the list in descending order.
