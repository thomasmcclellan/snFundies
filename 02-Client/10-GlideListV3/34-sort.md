# `sort(field)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| field | `string` | The field to be used to sort the list. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sort the list in ascending order.
