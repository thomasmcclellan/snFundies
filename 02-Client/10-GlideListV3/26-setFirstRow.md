# `setFirstRow(firstRow)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| firstRow | `number` | The row number in the list. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the first row to be displayed when the list is refreshed.
