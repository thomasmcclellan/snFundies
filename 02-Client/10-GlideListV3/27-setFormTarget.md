# `setFormTarget(target)`

## Parameter(s):
| Name | Type | Description |
|---|---|---|
| target | `string` | The `form.target` attribute value to use. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Specifies where to display the response from the form.
