# `render()`
## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Renders the UI page in the modal.

---

```js
var gm = new GlideModal('UI_dialog_name');
// Sets the dialog title
gm.setTitle('Show Title');
gm.setWidth(550);
// Opens the dialog
gm.render();
```
