# `orderBy(column)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| column | `string` | The column to add to sort the result set. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Specifies an `orderBy` column.  May be called more than once to order by multiple columns.
