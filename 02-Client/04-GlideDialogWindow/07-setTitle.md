# `setTitle(title)`

## Parameter(s):
| Name | Type | Description |
|---|---|---|
| title | `string` | The title for the current window. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the title of the dialog window.

---

```js
var gdw = new GlideDialogWindow('show_list');

gdw.setTitle('Test');
```
