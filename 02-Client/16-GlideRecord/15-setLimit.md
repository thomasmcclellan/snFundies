# `setLimit(maxQuery)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| maxQuery | `number` | The limit of the `number` of records to retrieve. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the limit fr how many records are in the `GlideRecord`.
