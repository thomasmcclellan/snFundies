# `setEnabled(item)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| item | `GlideMenuItem` | The item to be enabled. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Enables the specified menu item.

---

```js
g_menu.setEnabled(g_item);
```
