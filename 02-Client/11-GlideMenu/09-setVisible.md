# `setVisible(item)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| item | `GlideMenuItem` | This item to be displayed. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Displays a specific item.

---

```js
g_menu.setVisible(g_item);
```
