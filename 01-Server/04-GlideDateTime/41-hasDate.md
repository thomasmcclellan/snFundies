# `hasDate()`

## Return value:
| Type | Description |
|---|---|
| `boolean` | `true` if the object date is set; otherwise returns `false`. |

---

## Description:
Determines if an object's date is set.

---

```js
var gdt = new GlideDateTime('2019-09-10 09:00:00');

gs.info(gdt.hasDate());
```
