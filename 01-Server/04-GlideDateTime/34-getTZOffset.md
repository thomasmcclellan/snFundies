# `getTZOffset)`

## Return value:
| Type | Description |
|---|---|
| `number` | The number of milliseconds of time zone offset. |

---

## Description:
Gets the time zone offset in milliseconds.

---

```js
var gdt = new GlideDateTime();

gdt.getLocalTime();

gs.info(gdt.getTZOffset());
```
