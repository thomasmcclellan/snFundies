# `getYearNoTZ()`

## Return value:
| Type | Description |
|---|---|
| Number | The numerical value of the year. |

---

## Description:
Gets the year stored by the GlideDate object, expressed in the UTC time zone.

---

```js
// Today's date is 2019-07-24
var gd = new GlideDate();

gs.info(gd.getYearNoTZ()); // 2019
```
