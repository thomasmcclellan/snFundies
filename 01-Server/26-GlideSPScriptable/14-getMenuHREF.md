# `getMenuHREF(page)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| page | `GlideRecord` | The page. |

## Return value:
| Type | Description |
|---|---|
| `string` | The href portion of the URL. |

---

## Description:
Returns the `?id=` portion of the URL based on the `sp_menu` type.
