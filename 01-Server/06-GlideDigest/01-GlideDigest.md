# `GlideDigest()`
## Overview: 
The scoped `GlideDigest` class provides methods for creating a message digest from `strings` or input streams using `MD5`, `SHA1`, or `SHA256` hash algorithms.

---

## Description:
Creates an instance of scoped `GlideDigest`.
