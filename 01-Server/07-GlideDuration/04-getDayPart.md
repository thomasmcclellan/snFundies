# `getDayPart()`

## Return value:
| Type | Description |
|---|---|
| `number` | The number of days. |

---

## Description:
Gets the number of days.

---

```js
var dur = new GlideDuration('3 12:00:00');

gs.info(dur.getDayPart());
```
