# `getDurationValue()`

## Return value:
| Type | Description |
|---|---|
| `string` | The duration value. |

---

## Description:
Gets the duration value in 'd HH:mm:ss' format.

---

```js
var dur = ne GlideDuration('3 12:00:00');

gs.info(dur.getDurationValue());
```
