# `getTableName()`
## Return value:
| Type | Description |
|---|---|
| `string` | The table name. |

---

## Description:
Retrieves the name fo the table with the `GlideRecord`.

---

```js
var gr = new GlideRecord('incident');

gs.info(gr.getTableName());
```
