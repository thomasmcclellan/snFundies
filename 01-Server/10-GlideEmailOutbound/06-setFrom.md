# `setFrom(address)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| address | `string` | The sender's email address. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the sender's address.

---

```js
email.setFrom('joe.employee@something.com');
```
