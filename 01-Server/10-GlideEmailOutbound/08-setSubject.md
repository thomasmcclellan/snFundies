# `setSubject(subject)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| subject | `string` | Text for the subject line. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the email's subject line.

---

```js
email.setSubject('Important Issue to Discuss');
```
