# `setReplyTo(address)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| address | `string` | The reply to email address. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the reply to address.

---

```js
email.setReplyTo('joe.employee@something.com');
```
