# `setStatus(status)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| status | `number` | The status to be set. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the status code for the response.

---

```js
g_response.setStatus(200);
```
