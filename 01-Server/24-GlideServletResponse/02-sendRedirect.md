# `sendRedirect(location)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| location | `string` | The URL to receive the response. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sends a temporary redirect to the client.
