# `setContentType(type)`
## Parameter(s):
| Name | Type | Description |
|---|---|---|
| type | `string` | The `MIME` type. |

## Return value:
| Type | Description |
|---|---|
| Void | Method does not return a value. |

---

## Description:
Sets the `MIME` type of the response.

---

```js
g_response.setContentType('text/html;charset=UTF-8');
```
